ðŸ§  Inheritance and Prototypes
- Every class inherits from Object unless specified otherwise.
- You can inspect or modify inheritance using the base property.
- Static methods and properties belong to the class object; instance methods belong to the class's Prototype.
Example:
class MyClass {
    static StaticProp := "I'm static"
    InstanceProp := "I'm instance"
    Method() => MsgBox "Hello from Method"
}
obj := MyClass()
MsgBox obj.InstanceProp  ; Access instance property
MsgBox MyClass.StaticProp  ; Access static property
