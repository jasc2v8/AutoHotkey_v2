INNO SETUP COMMAND LINE:

"C:\Program Files (x86)\Inno Setup 6\ISCC.exe" /Q exeName.iss ; /Quiet (future log file?)

The basic syntax is:
iscc [options] <script name>

Common options include:

/Q: Quiet compile. It suppresses most output, showing only error messages.
/Qp: Quiet with progress. It displays a progress bar but no other messages.
/O<path>: Sets the output directory for the compiled installer. This overrides the OutputDir directive in the script.
/F<filename>: Specifies the output filename. This overrides the OutputBaseFilename directive in the script.
/S<name>=<command>: Defines a SignTool command to use for signing the installer.
/D<name>[=<value>]: Defines a preprocessor variable, useful for passing build-specific information into the script.
For example, to compile a script quietly and place the output in a specific folder, you would use:
iscc /Q /O"C:\MyBuilds" "C:\MyProject\MySetup.iss"
